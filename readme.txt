Thank you for downloading Yura Sashoune Gyrophare!

Gyrophare is a Petit Mot style edit of the Gyroflex French database

Info:

[a a.] = VV combination with glottal stop in between. For usage, add the point at the end of the [V V]

[a .] = glottal end stop

[. a] = glottal beginning

[a -] = end breath. Primarily used to make the endings of vowels softer

[a -hh] = end breath with exaggerated breathiness

[a -hhi] = end breath followed by a breathing in. Can be used to mimic the breathing in between words.
